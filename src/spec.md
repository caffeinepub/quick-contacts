# Specification

## Summary
**Goal:** Build a contact management application where users can add contacts by searching usernames, sharing invite links, or scanning QR codes.

**Planned changes:**
- Create backend user registry with unique usernames and contact storage
- Implement invite link generation and validation system
- Build username search interface with add contact functionality
- Add invite link generation and sharing with copy-to-clipboard
- Implement QR code generation for user profiles
- Create QR code scanner using device camera
- Display contacts list with profile information
- Design modern interface with warm earth tones (terracotta, sage green, cream), rounded corners, and friendly typography

**User-visible outcome:** Users can create profiles with unique usernames, find and add contacts through username search, generate and share invite links, display personal QR codes for others to scan, scan QR codes to instantly add contacts, and view their complete contacts list in a clean, approachable interface.
